Для установки платежного модуля kaznachey необходимо произвести следующие действия:
 
1. Скопируйте содержимое папки в корневой каталог сайта.

2. Выполните запросы из файла install.sql (установка платежного процессора и переводов модуля на русский и английский языки)
 
3. Войдите в раздел администрирования CS-Cart, перейдите в "Способы оплаты", выберете "Добавить способ оплаты".

   В разделе "Общее":
       Название - произвольное
       Процессор - kaznachey
       Статус - Вкл
   В разделе "Настроить":
       Merchant ID
       Секретный ключ
       Курс валюты по умолчанию к грн. (=1 если основная валюта - гривна)
